# 🎮 Games Collection by ThatOneGuy-67

Welcome to the Games Collection — a fun set of browser-based HTML games created by ThatOneGuy-67.  
These games are lightweight, interactive, and designed for anyone who wants to play or learn from simple web game projects.  

All games are standalone HTML files that run directly in your browser. Open a file and start playing instantly — no downloads or installations required. The collection is perfect for beginners who want to explore HTML, CSS, and JavaScript in a hands-on way, or for anyone looking for a quick, entertaining diversion.

---

## 🌟 About the Collection

This repository is a playground for experimenting with web games. Each game has its own unique mechanics and style.  
Some games are simple and relaxing, while others are more challenging and interactive. Every file is independent, meaning you can open and modify them without affecting the others.

The collection emphasizes:

- **Simplicity**: Easy to understand code for learning purposes.  
- **Accessibility**: Works in any modern browser.  
- **Customizability**: Modify graphics, sounds, or gameplay easily.  
- **Variety**: Games range from fun experiments to mini challenges.  

---

## 🚀 How to Play

1. Download or clone the repository.  
2. Open the folder containing the games.  
3. Double-click any `.html` file to open it in your web browser.  
4. Play and enjoy!

No additional tools or setup are required.

---

## 🛠️ Tech Used

- HTML5  
- CSS3  
- JavaScript (vanilla)  

Each game is self-contained and requires only a web browser to run.
